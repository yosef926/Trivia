# Trivia

A multiplayer trivia game built with a multithreaded C++ server and a C# WPF frontend. It uses socket programming to handle concurrent players, real-time lobby management, user auth, and live gameplay state updates smoothly.

## Features

* **Multithreaded Server:** Handles multiple concurrent player sessions efficiently without blocking the core game loop.
* **WPF Client Interface:** A clean, responsive Windows desktop GUI for an intuitive user experience.
* **Real-Time Lobbies:** Dynamic room creation, browsing, and joining mechanisms.
* **Live Game Sync:** Automated trivia loops, real-time question delivery, countdown timers, and instant score updates.
* **User Authentication:** Secure signup and login verification system.
